# LS45 AutoMotriz
Proyecto desarrollado como entrega de Examen Final de Diseño Web para la empresa **LS45 AutoMotriz**.
El sitio web fue diseñado con el objetivo de realizar un sitio web para la empresa antes mencionada, mejorando la navegación, la experiencia de usuario y la presentación visual de los servicios automotrices ofrecidos.

## Descripción del Proyecto
Este proyecto corresponde a la versión final de un sitio web corporativo desarrollado mediante la adaptación y personalización de la plantilla Carbook Master, modificando su estructura, contenido y estilos para ajustarse a las necesidades de LS45 AutoMotriz.

Entre las principales mejoras realizadas se encuentran:
- Adaptación completa del contenido.
- Rediseño visual.
- Optimización de navegación y estructura del sitio.
- Incorporación de secciones corporativas y de servicios.
- Personalización para LS45Automotriz.

## Tecnologías Utilizadas
